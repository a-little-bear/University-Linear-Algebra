在经典分析中，凸函数满足 $f(\lambda x + (1-\lambda)y) \le \lambda f(x) + (1-\lambda)f(y)$。然而，当我们把变量替换为矩阵时，情况变得极为复杂。**矩阵凸函数**（又称算子凸函数）要求这一不等式在 **Löwner 偏序**意义下成立。这一严格的约束剔除了一大批普通的凸函数，仅留下具有深层解析背景（如 Pick 函数）的一类算子。本章将探讨这种强大的非线性算子性质。

---

## 64B.1 算子凸性定义

!!! definition "定义 64B.1 (算子凸函数)"
    设 $f$ 是定义在区间 $I$ 上的实函数。若对于所有特征值位于 $I$ 内的对称矩阵 $A, B$，以及 $\lambda \in [0, 1]$，均满足：
    $$f(\lambda A + (1-\lambda)B) \preceq \lambda f(A) + (1-\lambda)f(B)$$
    则称 $f$ 在区间 $I$ 上是**算子凸的**（Operator Convex）。

!!! warning "算子凸 vs 标量凸"
    算子凸性蕴含标量凸性，但反之不成立。例如 $f(x) = x^4$ 是标量凸的，但它**不是**算子凸的。

---

## 64B.2 核心算子凸函数

!!! theorem "定理 64B.1 (典型算子凸函数)"
    1.  **逆函数**：$f(x) = 1/x$ 在 $(0, \infty)$ 上是算子凸的（且算子递减）。
    2.  **平方函数**：$f(x) = x^2$ 是算子凸的。
    3.  **算子熵**：$f(x) = x \log x$ 是算子凸的（量子信息论的基础）。
    4.  **负幂次**：$f(x) = x^r$ 在 $r \in [1, 2]$ 或 $r \in [-1, 0]$ 时是算子凸的。

---

## 64B.3 Jensen 算子不等式

!!! theorem "定理 64B.2 (Jensen 算子不等式)"
    设 $f$ 是算子凸函数，且 $\sum V_i^* V_i = I$（单位分解）。则：
    $$f\left( \sum V_i^* A_i V_i \right) \preceq \sum V_i^* f(A_i) V_i$$
    **应用**：这是证明量子信道数据处理不等式的核心工具。

---

## 64B.4 Kraus 定理

!!! theorem "定理 64B.3 (Kraus 定理)"
    一个函数 $f$ 是算子凸的，当且仅当它可以表示为以下积分形式：
    $$f(x) = a + bx + cx^2 + \int \frac{(x-\lambda)^2}{1+\lambda^2} d\mu(\lambda)$$
    其中 $c \ge 0$ 且 $\mu$ 是正测度。这说明算子凸性与复分析中的解析延拓有着本质联系。

## 练习题

1. **[基础] 证明 $f(x) = ax+b$ 既是算子凸的也是算子凹的。**

   ??? success "参考答案"
       由于 $f(\lambda A + (1-\lambda)B) = a(\lambda A + (1-\lambda)B) + bI = \lambda f(A) + (1-\lambda)f(B)$，等号恒成立。

2. **[反例] 验证 $f(x) = x^3$ 在正定阵上是否为算子凸函数。**

   ??? success "参考答案"
       不是。虽然它在 $\mathbb{R}^+$ 上是标量凸的，但在矩阵层面不满足算子凸性。

3. **[性质] 证明：若 $f$ 是算子凸的，则 $g(x) = f(x) + c$ 也是算子凸的。**

   ??? success "参考答案"
       常数项在不等式两边抵消，不影响 Löwner 序的关系。

4. **[逆函数] 解释为什么 $A \preceq B \implies B^{-1} \preceq A^{-1}$。**

   ??? success "参考答案"
       这是 $f(x) = 1/x$ 作为算子单调递减函数的直接结论。

5. **[组合] 两个算子凸函数的和是算子凸的吗？**

   ??? success "参考答案"
       是的。Löwner 序对加法是封闭的。

6. **[应用] 在量子力学中，为什么 von Neumann 熵 $S(\rho) = -\operatorname{tr}(\rho \log \rho)$ 是凹的？**

   ??? success "参考答案"
       因为 $f(x) = x \log x$ 是算子凸函数，取负号后变为算子凹，迹运算保持这一凹性。

7. **[单调性] 算子凸函数一定是算子单调的吗？**

   ??? success "参考答案"
       不一定。例如 $x^2$ 是算子凸的，但在 $(0, \infty)$ 上不是算子单调的（除非限制在特定的正定域内）。

8. **[计算] 若 $A = \operatorname{diag}(1, 2), B = \operatorname{diag}(2, 1)$，验证 $f(x)=x^2$ 的算子凸性。**

   ??? success "参考答案"
       $(\frac{A+B}{2})^2 = (1.5 I)^2 = 2.25 I$。$\frac{A^2+B^2}{2} = \frac{\operatorname{diag}(1, 4) + \operatorname{diag}(4, 1)}{2} = 2.5 I$。显然 $2.25 I \preceq 2.5 I$。

9. **[SSA] 什么是量子熵的强次可加性 (SSA)？**

   ??? success "参考答案"
       $S(\rho_{ABC}) + S(\rho_B) \le S(\rho_{AB}) + S(\rho_{BC})$。其代数证明核心就是利用 $x \log x$ 的算子凸性及其导出的迹不等式。

10. **[极限] 为什么算子凸性要求函数必须是 $C^2$ 连续的？**

   ??? success "参考答案"
        事实上，算子凸性比 $C^2$ 更强，它要求二阶 Fréchet 导数作为双线性型必须是正定的（即 $f$ 的二阶差商矩阵必须正定）。

## 本章小结

矩阵凸函数确立了非线性算子作用下的稳定性准则：

1.  **序的刚性**：算子凸性证明了在非交换环境下，保持偏序关系的非线性变换是极其罕见的，这为物理系统的能量演化设定了严格的数学边界。
2.  **解析的纽带**：Kraus 和 Löwner 定理将纯代数的不等式与复平面上的解析性质挂钩，揭示了算子凸性本质上是某种全纯结构的投影。
3.  **信息的基石**：以 $x \log x$ 为代表的算子凸函数支撑了整个量子信息论的熵增原理与稳定性分析，证明了代数的凸性即是宇宙信息的守恒逻辑。
