线性保持问题 (LPP) 涉及刻画使某些性质、子集或关系保持不变的线性映射 $\phi: M_n \to M_n$。这一领域由 Frobenius 于 1897 年开创，揭示了矩阵结构的“刚性”：保持核心性质（如行列式或秩）的映射通常必须具有非常受限的形式，如相似变换或转置。

---

## 60.1 Frobenius 与 Marcus-Moyls 定理

!!! theorem "定理 60.1 (Frobenius 行列式保持定理, 1897)"
    设 $\phi: M_n(\mathbb{C}) \to M_n(\mathbb{C})$ 为线性映射，且满足对所有 $A$ 都有 $\det(\phi(A)) = \det(A)$。则存在满足 $\det(MN) = 1$ 的可逆矩阵 $M, N$ 使得：
    $$\phi(A) = MAN \quad \text{或 } \quad \phi(A) = MA^T N$$

!!! theorem "定理 60.3 (Marcus-Moyls 秩-1 保持定理)"
    保持秩-1 矩阵集合的线性映射必须具有形式 $\phi(A) = MAN$ 或 $\phi(A) = MA^T N$。由于秩-1 矩阵是矩阵空间的“基石”，许多 LPP 结果都可以归约到此定理。

## 练习题

1. **[概念] 定义线性保持问题中的“标准形式”。**
   ??? success "参考答案"
       标准形式是指形如 $\phi(A) = MAN$（类相似变换）和 $\phi(A) = MA^T N$（类转置变换）的映射。大多数 LPP 结果证明，保持特定代数不变量的映射必然落入这两类。

2. **[迹保持] 证明转置映射 $\phi(A) = A^T$ 保持迹和行列式。**
   ??? success "参考答案"
       $\operatorname{tr}(A^T) = \sum (A^T)_{ii} = \sum a_{ii} = \operatorname{tr}(A)$。对于行列式，$\det(A^T) = \det(A)$ 是行列式理论的基本恒等式。转置是非平凡 LPP 的原型。

3. **[谱保持] 证明相似变换 $\phi(A) = PAP^{-1}$ 是谱保持的。**
   ??? success "参考答案"
       特征多项式满足 $\det(\lambda I - PAP^{-1}) = \det(P(\lambda I - A)P^{-1}) = \det(P) \det(\lambda I - A) \det(P)^{-1} = \det(\lambda I - A)$。相同的特征多项式意味着相同的特征值集合。

4. **[秩-1] 为什么保持秩-1 矩阵被认为是解决许多 LPP 的关键步骤？**
   ??? success "参考答案"
       秩-1 矩阵是 PSD 锥的极端射线，也是矩阵流形中最简单的非零元素。许多性质（如秩、行列式次数、图像子空间）都由映射在这些基础元素上的行为决定。

5. **[可逆性] 叙述关于保持奇异矩阵的 Dieudonné 定理。**
   ??? success "参考答案"
       一个将奇异矩阵集合映射到自身的线性双射必须具有形式 $\phi(A) = MAN$ 或 $\phi(A) = MA^T N$。这表明可逆性是一种“刚性”性质，限制了空间的对称群。

6. **[正映射] 定义“正映射”并提供一个不是相似变换的例子。**
   ??? success "参考答案"
       若 $A \succeq 0 \implies \phi(A) \succeq 0$，则称 $\phi$ 为正映射。转置映射 $\phi(A) = A^T$ 是正映射，但不是相似变换。

7. **[完全正性] 区分正映射与完全正 (CP) 映射。**
   ??? success "参考答案"
       若对于所有 $k \ge 1$，扩展映射 $\phi \otimes \operatorname{id}_k$ 都是正映射，则称 $\phi$ 为完全正映射。转置是正映射但不是 CP 映射（它在纠缠空间中会产生负特征值），而 $\phi(A) = VAV^*$ 是 CP 映射。

8. **[Choi定理] CP 映射的 Kraus 表示是什么？**
   ??? success "参考答案"
       $\phi(A) = \sum V_i A V_i^*$。这种算子和形式确保了即使系统耦合到任意环境，映射仍保持正性。

9. **[交换性] 描述 $n \ge 3$ 时保持交换性的线性映射的形式。**
   ??? success "参考答案"
       此类映射必须满足 $\phi(A) = \alpha P A P^{-1} + f(A)I$ 或 $\phi(A) = \alpha P A^T P^{-1} + f(A)I$，其中 $f$ 是线性泛函。项 $f(A)I$ 反映了单位阵与所有矩阵对易，从而充当了中心。

10. **[量子力学] 解释 LPP 结果与量子信道的相关性。**
    ??? success "参考答案"
        量子信道在数学上被建模为 CPTP（完全正且保迹）映射。LPP 理论刻画了密度矩阵所允许的物理变换，并定义了保持纠缠的算子结构。

## 本章小结

本章探讨了矩阵代数在保持特定性质下的刚性：

1. **基础定理**：Frobenius 和 Marcus-Moyls 定理确立了保持行列式和秩的映射的标准形式。
2. **谱刚性**：证明了谱保持映射本质上仅限于相似变换和转置。
3. **正性等级**：探讨了正映射与完全正映射的层次结构，为量子操作提供了数学基础。
4. **结构对称性**：揭示了保持交换性或可逆性等基本代数关系会强制映射进入高度结构化的几何形式。
