在热带半环上，加法是取最小值，乘法是加法。当我们在这个奇特的数域上定义多项式并研究其零点时，便诞生了**热带几何**（Tropical Geometry）。在这里，圆滑的代数曲线变成了锐利的、“多刺”的线性图形。本章将展示热带几何如何通过将代数问题转化为图论和凸几何问题，为现代几何学提供了一套强大的计算工具。

---

## 59B.1 热带多项式与超曲面

!!! definition "定义 59B.1 (热带多项式)"
    变量 $\mathbf{x} = (x_1, \ldots, x_n)$ 的热带多项式是形式为 $F(\mathbf{x}) = \bigoplus_i (c_i \otimes \mathbf{x}^{\mathbf{a}_i})$ 的表达式。在经典算术下，它是一个分段线性凸函数：
    $$f(x_1, \ldots, x_n) = \min_i \{ c_i + a_{i1}x_1 + \cdots + a_{in}x_n \}$$

!!! definition "定义 59B.2 (热带超曲面)"
    热带多项式 $F$ 定义的**热带超曲面** $V(F)$ 是函数 $f(\mathbf{x})$ 的**不可导点集**（即最小值在至少两个项处同时达到的点集）。

---

## 59B.2 Amoeba 与热带化

!!! technique "Amoeba (阿米巴变形虫)"
    对于复代数簇 $V \subset (\mathbb{C}^*)^n$，其 **Amoeba** 是在映射 $\operatorname{Log}(z_1, \ldots, z_n) = (\ln|z_1|, \ldots, \ln|z_n|)$ 下的像。
    **热带化**：当对数基数趋于无穷大时，Amoeba 会收缩为一个由线段构成的骨架，这个骨架就是**热带簇**。

---

## 59B.3 基本定理：Kapranov 定理

!!! theorem "定理 59B.1 (Kapranov 定理)"
    对于由 Laurent 多项式 $f = \sum c_w z^w$ 定义的超曲面，其热带化恰好是由热带多项面 $F = \bigoplus \operatorname{val}(c_w) \otimes x^w$ 定义的热带超曲面。这确立了经典代数几何与热带几何之间的精确对应。

---

## 59B.4 热带直线

!!! example "热带直线"
    在 $\mathbb{R}^2$ 中，热带直线 $L(x, y) = 0 \oplus x \oplus y$ 的图像是三个从原点出发的射线，方向分别为 $(-1, -1)$、$(1, 0)$ 和 $(0, 1)$。它们满足“平衡条件”，即这些方向向量之和为零。

## 本章小结

热带几何是代数几何的“极低能耗”投影：


****：它将指数级复杂的代数方程坍缩为线性的骨架，保留了拓扑核心（如交点数、亏格），极大简化了计算。

****：通过 Amoeba 的极限过程，热带几何证明了离散的组合结构（图、扇形复形）是连续代数形态的终极本质。

****：作为解决枚举几何难题的利器，热带几何展示了如何通过“数线条”来解决“求方程”的难题，为现代数学物理提供了新的直观。
