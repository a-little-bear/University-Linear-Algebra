## 目录与学习路径

### [第一部分：基础线性代数](./part1/ch01-linear-equations.md) { .difficulty-tag .beginner }
*线性代数的基石：方程组、矩阵运算、行列式与向量空间基础。*

### [第二部分：中级线性代数](./part2/ch08-inner-product-spaces.md) { .difficulty-tag .intermediate }
*抽象结构的进阶：内积空间、二次型、商空间与有理标准形。*

### [第三部分：高级矩阵分析](./part3/ch14-matrix-analysis.md) { .difficulty-tag .advanced }
*分析与理论的巅峰：算子范数、谱理论、矩阵函数与矩阵方程。*

### [第四部分：特殊矩阵结构](./part6/ch33-generalized-inverses.md) { .difficulty-tag .advanced }
*特殊算子的深挖：广义逆、Schur补、Toeplitz矩阵与全正矩阵。*

### [第五部分：代数结构推广](./part4/ch21-multilinear-algebra.md) { .difficulty-tag .research }
*现代数学的语言：多线性代数（张量）、外代数、Clifford代数与有限域。*

### [第六部分：数值与随机专题](./part4/ch22-numerical-linear-algebra.md) { .difficulty-tag .research }
*计算与不确定性：数值算法、随机矩阵理论与集中不等式。*

### [第七部分：矩阵几何与优化](./part4/ch24-matrix-manifolds.md) { .difficulty-tag .research }
*几何视角下的优化：矩阵流形、特征值逆问题与凸分析。*

### [第八部分：组合与代数专题](./part8/ch65a-sign-patterns.md) { .difficulty-tag .research }
*跨学科的纽带：符号模式、热带几何与同时三角化。*

### [第九部分：应用全览](./part5/ch26-differential-equations.md) { .difficulty-tag .intermediate }
*线性代数的威力：控制理论、量子信息、机器学习、机器人学与生物学。*
