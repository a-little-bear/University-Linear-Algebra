如果说矩阵是线性代数的骨架，那么向量空间就是它的灵魂。通过将具体的计算规则抽象为公理，我们获得了一种处理一切满足线性叠加原理的对象的通用方法。本章将从严谨的定义出发，逐步建立起描述空间的“骨架”——基与维数。

---

## 04.1 向量空间与公理

!!! definition "定义 04.1 (向量空间)"
    一个集合 $V$ 称为域 $F$ 上的**向量空间**，如果其上定义了加法和数乘运算，并满足以下 8 条公理：
    1.  **加法结合律与交换律**。
    2.  **存在零向量 $\mathbf{0}$**。
    3.  **存在负向量**。
    4.  **数乘结合律**。
    5.  **单位标量作用**：$1\mathbf{v} = \mathbf{v}$。
    6.  **标量分配律**：$c(\mathbf{u}+\mathbf{v}) = c\mathbf{u} + c\mathbf{v}$。
    7.  **向量分配律**：$(a+b)\mathbf{v} = a\mathbf{v} + b\mathbf{v}$。

---

## 04.2 子空间与张成

!!! definition "定义 04.2 (子空间)"
    $V$ 的非空子集 $W$ 称为**子空间**，如果 $W$ 对加法和数乘**封闭**：
    - 若 $\mathbf{u}, \mathbf{v} \in W$，则 $\mathbf{u} + \mathbf{v} \in W$。
    - 若 $\mathbf{u} \in W, c \in F$，则 $c\mathbf{u} \in W$。

!!! definition "定义 04.3 (张成空间 Span)"
    由向量组 $\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$ 的所有线性组合构成的集合称为它们的**张成空间**：
    $$\operatorname{span}\{\mathbf{v}_1, \ldots, \mathbf{v}_k\} = \{c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k : c_i \in F\}$$

---

## 04.3 基与维数

!!! definition "定义 04.4 (线性相关与无关)"
    若 $c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$ 仅在 $c_i = 0$ 时成立，则向量组**线性无关**。

!!! definition "定义 04.5 (基与维数)"
    若向量组 $B = \{\mathbf{b}_1, \ldots, \mathbf{b}_n\}$ 线性无关且张成 $V$，则称 $B$ 是 $V$ 的一个**基**。基中向量的个数称为 $V$ 的**维数** $\dim(V)$。

---

## 04.4 矩阵的四个基本子空间

!!! theorem "定理 04.1 (四个基本子空间)"
    对于 $m \times n$ 矩阵 $A$：
    1.  **列空间 $C(A)$**：$A$ 的列张成的空间，在 $\mathbb{R}^m$ 中。
    2.  **零空间 $N(A)$**：满足 $Ax = \mathbf{0}$ 的所有 $x$ 构成的空间，在 $\mathbb{R}^n$ 中。
    3.  **行空间 $C(A^T)$**：$A$ 的行张成的空间，在 $\mathbb{R}^n$ 中。
    4.  **左零空间 $N(A^T)$**：满足 $A^T y = \mathbf{0}$ 的所有 $y$ 构成的空间，在 $\mathbb{R}^m$ 中。

!!! theorem "定理 04.2 (秩-零化度定理)"
    $\dim C(A) + \dim N(A) = n$（列数）。即：**秩 + 零化度 = 总列数**。

    ## 本章小结

向量空间确立了线性代数的几何与代数底座：


****：通过公理化，我们能将不同领域的数学对象（多项式、信号）视为同一类实体（向量）进行处理。

****：基与维数的概念量化了空间的“容量”与“朝向”，确立了描述对象的最小坐标系。

****：四个基本子空间的划分揭示了线性算子 $A$ 内部最深层的构造，以及输入输出之间维数守恒的必然性。
